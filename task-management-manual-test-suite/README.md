## Test Case Structure

Each test case includes the following fields:

| Field | Description |
|---|---|
| Test Case ID | Unique identifier |
| Title & Module | Test name and module it belongs to |
| Test Category | Type of testing performed |
| Preconditions | Required state before execution |
| Test Steps | Step-by-step execution instructions |
| Test Data | Input values used |
| Expected Result | What should happen |
| Actual Result | What actually happened |
| Priority & Severity | Importance and impact level |
| Status & Execution Date | Pass/Fail and date of execution |
